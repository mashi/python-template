# Description
Template repository for python projects.


# Instructions
## How to use this repository
Click on the "use this template button".


## Development
On your local machine, run
```
make install
```

To run tests:
```
make tests
```

To generate the documentation:
```
make docs
```

and open the `docs/_build/index.html` file.
